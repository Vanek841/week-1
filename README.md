### Что я уже освоил в Bandit

- Базовая работа с терминалом (`ls`, `cd`, `cat`, `pwd`)

- `grep`, `sort | uniq -u`, `strings`, `base64 -d`
- Перенаправление вывода и подавление ошибок (`2>/dev/null`)
